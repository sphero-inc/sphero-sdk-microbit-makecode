# set_undercarriage_white_led

```sig
spheroRvr.set_undercarriage_white_led(127);
```

## Parameters

* **intensity**: The brightness of the LED.  0 is off, 255 is full brightness.

## Example

```blocks
spheroRvr.set_undercarriage_white_led(127);
```
