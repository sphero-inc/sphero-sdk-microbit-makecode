# set_all_leds

```sig
spheroRvr.set_all_leds(95, 158, 160);
```

## Parameters

* **red**: The red value to which all LEDs will be set.
* **green**: The green value to which all LEDs will be set.
* **blue**: The blue value to which all LEDs will be set.

## Example

```blocks
spheroRvr.set_all_leds(95, 158, 160);
```
