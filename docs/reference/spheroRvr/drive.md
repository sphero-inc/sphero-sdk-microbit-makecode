# drive

```sig
spheroRvr.drive(heading, speed)
```

## Parameters

* **heading**: The direction you want to drive.
* **speed**: The speed you want to travel.  Use negative values to go in reverse!

## Example

```blocks
spheroRvr.drive(135, -100);
```
